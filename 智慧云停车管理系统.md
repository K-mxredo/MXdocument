[TYPE] 

Sensitive Data Exposure

[DESCRIPTION] 

A Sensitive Data Exposure  vulnerability exists in the "智慧云停车管理平台 ",.The system's API is vulnerable to unauthorized access, allowing unauthenticated users to exploit certain interfaces and retrieve  sensitive user data  posing a critical security risk.

[DETAILS] 

1.A Sensitive Data Exposure vulnerability has been identified in the  application. The vulnerability allows attackers to exploit certain  interfaces and retrieve sensitive user data such as names, and passwords, posing a critical security risk.

The system's API is vulnerable to unauthorized access,use  "parking-system/provider/user/id" allows retrieval of detailed personnel data  without logging in。

![图片](https://github.com/user-attachments/assets/4054eee6-8f31-4ef3-b0cf-5ca42780068e)


2.By iterating through the `id` parameter, you can retrieve the personal information of all users.
![图片](https://github.com/user-attachments/assets/3d1c7d83-49cd-48d7-9f26-8622a726a7ed)



[Mitigation & Fix Recommendations] 

Unauthorized access is prohibited. Implement server-side authorization checks to enforce access control.
