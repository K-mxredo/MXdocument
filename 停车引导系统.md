Arbitrary File Read Vulnerability

[DESCRIPTION] 

A Arbitrary File Read  vulnerability exists in the "车位引导管理系统 ".The system initiates a request to the server when loading parking lot images. This request fails to filter special characters, resulting in an arbitrary file read vulnerability.

[DETAILS] 

1.The system's API  "ParkConfig/GetMapByMapId" retrieves the map URL and then automatically loads it to fetch the map data.
